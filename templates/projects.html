A selection of apps I've created to help me develop my skills but also help me in day-to-day life.

Speed Test Tweeter

Created as a project to hone my skills with Python and Linux.

Utilising a Raspberry Pi I've setup a Python application to run a speed test API three times daily using built-in Linux task scheduler Cron. It then writes the test results to a CSV file and tweets them to my bot account using the Twitter API. If the speed isn't as promised by my ISP a different tweet is posted.

Amazon Price Checker

This project allowed me to practice Web Scraping with Beautiful Soup 4 and email with SMTPLIB.

It checks daily the price of a product on Amazon and if it's below a certain price I receive an email alerting me which also contains the page link for fast purchase.

Will It Rain Today?

One of the first projects I completed that utilizes two different API's and sends SMS messages.

Using Python3 this app checks the OpenWeatherMap API for the latest weather forcast in my area and if it's going to rain, It sends me an SMS message using the Twilio API.
