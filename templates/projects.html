A selection of apps I've created to help me develop my skills but also help me in day-to-day life.

Speed Test Tweeter


Created as a project to hone my skills with Python and Linux.

Utilising a Raspberry Pi I've setup a Python application to run a speed test API three times daily using built-in Linux task scheduler Cron. It then writes the test results to a CSV file and tweets them to my bot account using the Twitter API.
If the speed isn't as promised by my ISP a different tweet is posted.
